## 快速示例

**你说**: 帮我看看今天今日头条的热榜

**AI 自动完成**:
1. 找到热榜数据 API
2. 抓取 JSON 数据
3. 按热度排序
4. 输出结构化中文摘要

**输出效果**:
```
1️⃣ 四姑娘山大雪难挡"冲顶大军" [热度: 2065万]
2️⃣ 国内油价将迎调整 ⚡新 [热度: 1869万]
3️⃣ 从三个场景看"五一"出行服务新升级 [热度: 1691万]
...
```
