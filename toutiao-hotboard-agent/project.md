# 🗞️ 今日头条热榜自动抓取与总结 Agent

> **AI 小项目：让 AI Agent 自动浏览网页并提炼热榜摘要**

## 📌 项目简介

一个基于 **自然语言对话驱动** 的智能体（AI Agent）项目，用户只需说一句话（如"帮我看看今天头条热榜"），AI 便能：
1. **自主发现** → 通过已知站点结构找到热榜数据接口
2. **爬取内容** → 调用网页抓取工具获取原始 JSON 数据
3. **提炼摘要** → 解析数据、按热度排序、输出易读的中文热榜总结

完全不需要用户提供 URL 或手动操作浏览器。

---

## 🧠 核心流程（Agent 工作逻辑）

```
用户提问
    ↓
[AI Agent] 识别意图 → "需要看今日头条热榜"
    ↓
[AI Agent] URL 发现 → 已知头条热榜 API: /hot-event/hot-board/
    ↓
[AI Agent] 抓取数据 → Web Fetch 工具获取 JSON
    ↓
[AI Agent] 数据解析 → 提取 Title / HotValue / Label 等字段
    ↓
[AI Agent] 结构化输出 → 按热度降序 → 中文总结 + 分类标注
```

### 关键技术点

| 环节 | 做法 |
|------|------|
| **URL 发现** | 凭经验认知站点固定接口模式（无需搜索引擎） |
| **数据获取** | `web_fetch` 工具 + JSON 模式提取 |
| **内容截断处理** | 分上限获取、提取关键字段、舍弃冗余元数据 |
| **格式转换** | 原始 JSON → 中文自然语言榜单（保留序号 + 热度排序 + 标签） |
| **失败回退** | 404 时自动切换备选 URL / 接口路径 |
